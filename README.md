# Bot Discord HTW 🤖

Este es el bot oficial de Discord para la comunidad de la Hack The World. Está diseñado para supervisar la actividad de nuestros participantes e indicar los recordatorios indicados.

---
## ✨ ¿Qué puede hacer el bot?

Por ahora, estas son las funciones principales que tiene integradas:

* 📊 **Gestión de roles:** Permite a los usuarios asignarse sus propias materias o canales.
* 📚 **Info HTW:** Comandos rápidos con enlaces útiles de la universidad.
* ⚙️ **Moderación:** Herramientas básicas para mantener el orden en el servidor.
